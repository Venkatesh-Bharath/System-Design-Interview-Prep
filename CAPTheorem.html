CAP Theorem Triangle

📌 CAP Theorem — In a distributed system, you can only guarantee two out of three:

✅ C (Consistency) — All nodes see the same data at the same time

✅ A (Availability) — Every request receives a response (even if stale)

✅ P (Partition Tolerance) — System continues working despite network failures

💡 In real-world distributed systems, P is mandatory — you choose between C and A.
